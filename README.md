# Odin Recipes

A simple webpage that showcases recipes built as part of The Odin Project Foundations Course.

## Live Preview

https://bkill9.github.io/odin-recipes/index.html

## Project Objectives

- Practice writing HTML
- Create multiple webpages
- Use various elements such as headings, images, paragraphs, lists, and links

## Lessons Learned

- Structuring HTML pages
- Working with images (sizing large images down and writing good alt text)
- Organizing files and directories
